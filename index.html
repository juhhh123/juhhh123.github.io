Linux课程学习笔记

  • Linux 就是一个操作系统，就像你多少已经了解的 Windows（xp，7，8）和 Mac OS 。我们的 Linux 主要是系统调用和内核那两层。当然直观地看，我们使用的操作系统还包含一些在其上运行的应用程序，比如文本编辑器、浏览器、电子邮件等。
  • Linux命令行
    • ls命令-查找想要目录里的内容
    • pwd命令-查找目录所在的绝对路径
    • useradd命令-添加用户
    • passwd命令-修改密码
    • cd命令-切换到需要访问的目录
    • cp命令-复制文件和文件夹
    • rm命令-删除文件和目录
    • mv命令-移动文件和目录
    • touch命令-创建新的空文件
    • cat命令-查看文件
    • mkdir命令-创建文件夹和目录
    • su [usernane]命令-切换用户
    • help或man或info命令-帮助命令
    • date命令-显示或设置系统时间
    • cal [month] [year]-在屏幕上输出日历时间
    • clear命令-清楚终端中的信息
    • less命令-检查文件前后
    • df命令-使用df查看磁盘容量
    • htop命令-查看管理计算机的资源
  • Linux文件打包与解压缩
    • zip：
      • 打包：zip something.zip something(目录请加-r参数)
      • 解包：unzip something.zip
      • 指定路径：-d参数
    • tar：
      • 打包：tar -cf something.tar something
      • 解包：tar -xf something.tar
      • 指定路径：-C参数
  • Vim模式 vim 启动进入普通模式，处于插入模式或命令行模式时只需要按 Esc 或 即可进入普通模式。普通模式中按 i（插入）或 a（附加）键都可以进入插入模式，普通模式中按 : 进入命令行模式。命令行模式中输入 wq 回车后保存并退出 vim。
  • Linux网络管理及服务
    -ping使用ICMP协议来检测整个网络连通情况
    -traceroute 是数据包路由跟踪诊断命令，可以查看数据包在网络 上传输的路径情况
    -netstat -a显示已经建立连接的接口
    -dig 域信息搜集器
    -sudo apt install httpd安装服务器
  • Shell编程 shell 是一个命令解释器，是介于操作系统内核与用户之间的一个绝缘层。准确地说，它也是能力很强的计算机语言，被称为解释性语言或脚本语言。Bash（GNU Bourne-Again Shell）是一个为 GNU 计划编写的 Unix shell，它是许多 Linux 平台默认使用的 shell。
    -使用shell来执行sh hello.sh
    -使用bash来执行bash hello.sh
    -使用.来执行. ./hello.sh
